# Cheese-Escape

## Descripción del Juego

"Cheese-Escape" es un juego desarrollado en Unity donde asumes el papel de un valiente ratón. Tu misión principal es recolectar **5 trozos de queso** esparcidos por el nivel para asegurar la victoria.

Pero ten cuidado, el mundo está lleno de peligros:

*   **Enemigo Perseguidor**: Una criatura implacable que te perseguirá sin descanso. Si te atrapa, la partida terminará.
*   **Enemigo Tímido**: Una criatura no hostil que, al verte, intentará huir. No representa una amenaza directa, pero puede ser un obstáculo.

## Controles

El juego utiliza el sistema de entrada de Unity para una experiencia de control fluida.

*   **Movimiento**: Usa las teclas `W`, `A`, `S`, `D` o las flechas para mover al ratón.
*   **Salto**: Presiona la `Barra Espaciadora` para saltar.

## Objetivo

Recolecta los 5 quesos antes de ser alcanzado por el enemigo perseguidor. ¡Buena suerte, pequeño ratón!

## Desarrollo

Este proyecto está desarrollado con **Unity 6** .
